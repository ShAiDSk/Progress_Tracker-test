# âš™ï¸ Installation

### 1. Clone the repo  
```bash
git clone https://github.com/yourusername/progress-tracker.git
cd progress-tracker
```

### 2. Install dependencies

```bash
composer install
npm install
```

### 3. Environment setup

```bash
cp .env.example .env
php artisan key:generate
```

Update database details inside `.env`.

### 4. Run migrations

```bash
php artisan migrate
```

### 5. Build frontend

```bash
npm run dev
```

### 6. Start server

```bash
php artisan serve
```
